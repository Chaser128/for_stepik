## 👤 О персонаже

Я инженер технического сопровождения, который постепенно прокачивается в сторону **Data Engineering**.

В работе занимаюсь автоматизацией процессов, обработкой данных, поиском ошибок и созданием выгрузок.  
Люблю не просто исправлять проблему, а понимать:

- почему она появилась;
- где именно сломалась логика;
- как сделать так, чтобы она больше не повторилась.

```python
class Engineer:
    def __init__(self):
        self.role = "Technical Support Engineer"
        self.goal = "Data Engineer"
        self.energy = 100
        self.coffee = True

    def solve_problem(self, problem):
        while not problem.is_solved:
            problem.read_logs()
            problem.check_database()
            problem.test_hypothesis()

        return "Проблема решена ✅"

```
